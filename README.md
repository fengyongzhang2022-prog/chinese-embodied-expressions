# 中文里的身体密码 · Chinese Embodied Expressions

> A clean, interactive guide to 20 common embodied Chinese expressions for beginner–intermediate learners.

🌐 **Live Demo:** https://fengyongzhang2022-prog.github.io/chinese-embodied-expressions/

---

## 这是什么？ / What is this?

本项目是一个双语中文学习页面，从身体感觉出发，帮助学习者掌握 20 个地道、常用的中文具身表达（如 **心疼、头大、吃醋、走弯路** 等）。每个表达都配有：

- 真实场景故事
- 自然英语对照
- 例句 + 拼音 + 朗读
- “直译陷阱”提示
- 闪卡复习 + 情境测验

This is a bilingual Chinese learning page that teaches 20 common embodied expressions (e.g. **心疼, 头大, 吃醋, 走弯路**) through real-life contexts, natural English, flashcards, and quizzes.

---

## 核心功能 / Features

| 功能 | 说明 |
|------|------|
| **分类浏览** | 5 大感觉域：躯体疼痛、视觉味觉、温度触觉、空间重量、气流呼吸 |
| **搜索过滤** | 快速查找想学的表达 |
| **双语切换** | 一键显示/隐藏英文翻译 |
| **专注模式** | 隐藏解释，先猜后看，强化记忆 |
| **闪卡复习** | 翻转卡片，先抓关键词再确认含义 |
| **情境测验** | 15 道场景题，把表达放进真实语境中选择 |
| **语音朗读** | 点击 🔊 收听词语和例句发音 |
